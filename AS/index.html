Descobrindo qual a nota do filme Coringa

Sabemos que é dificil dar uma nota para um filme, ainda mais um filme como o Coringa. Então pensamos em criar uma maneira mais fácil para isso, dando notas para as cenas mais marcantes e lembradas pelo público. Após colocar uma nota de 0 até 10 para cada cena, iremos somar as notas, dividir pelo número de avaliações e assim trazer uma nota final para o filme.
